Новинки магазина

Новинки содержит информацию о новых товарах, поступивших в продажу. Упорядочивается по дням поступления товаров. Сортируется по коллекциям, затем по ID товара. Заглавная часть блока о новинках, должна содержать расширенную информацию об одном, определенном, наиболее интересном, с точки зрения администратора товаре. Остальные товары в блоке имеют краткую информацию. Эта страница должна создаваться автоматически при вводе новых товаров и сохраняться в дальнейшем. Содержание этой стр. необходимо дублировать в рассылке писем с новинками магазина. В Б.Д. хранится 200 подобных блоков, остальные автоматически удаляются, что бы не раздувать саму базу. Пример: http://rockmarket.ppd.spb.ru/incoming/
